# Amazon BSR 竞对分析 Skill

> 适用于 OpenClaw / 任意支持自定义 Skill 的 AI 工具
> 卖家精灵 / ASINking BSR 导出专用 · 输入 Top50 数据，输出竞争策略报告

---

## 解决什么问题

| 痛点 | Skill 做什么 |
|------|------------|
| 看完 50 条数据不知道市场结构是什么 | 自动还原「哑铃型/双寡头/分散型」格局 |
| 不知道哪个价格段值得切入 | 价格带地图标注竞争强度和机会评级 |
| 不清楚竞争壁垒有多高 | 五维度逐项打分，给出综合进入难度 |
| 差异化方向靠感觉 | 从差评/评价数/价格空白挖具体机会 |
| 不知道中国卖家在哪些地方有优势/弱点 | CN 卖家专项分析，找真空地带 |
| 选品策略不落地 | 输出 A/B 两套策略 + 明确不建议切入的区间 |

---

## 功能覆盖

| 模块 | 输出内容 |
|------|---------|
| 市场总览 | 规模 / 格局 / 集中度 / 新品占比 |
| 价格带竞争地图 | 款数 + 销售额占比 + 🔴🟠🟡🟢竞争强度 |
| 品牌格局 | 🏆⚔️🎯三类定位分析，附壁垒和弱点 |
| 竞争壁垒评估 | 评价/价格/时间/Listing/渠道五项，综合难度 X/10 |
| 差异化机会 | 价格空白 / 评分弱点 / 新品窗口 / Listing薄弱点 |
| 中国卖家分析 | 数量 / 销量 / 销售额占比，找竞争真空 |
| 选品切入策略 | A/B 策略卡片 + 明确不建议切入区间 |
| 核心结论 | 最大机会 / 最大风险 / 建议切入时间窗口 |

## 快速上手

### 第一步：加载 Skill

将 `Skill_BSR竞对分析_v1.md` 全文复制，粘贴到 AI 工具的 Skill / System Prompt 配置中。

### 第二步：从卖家精灵导出数据

```
卖家精灵 → 选品 → BSR 榜单
→ 选择类目和国家 → 导出 Top50 → Excel
```

核心字段：品牌 / 价格 / 月销量 / 评分数 / 上架天数
完整字段：以上 + 卖家所属地 / 留评率 / 毛利率 / LQS / 变体数

### 第三步：粘贴数据，发送指令

```
开始分析
```

## 符号说明

| 符号 | 含义 |
|------|------|
| 🔴 | 竞争壁垒极高 / 高风险 |
| 🟠 | 竞争激烈 / 需要差异化 |
| 🟡 | 机会点 / 可切入 |
| 🟢 | 低竞争 / 窗口期 |
| 🏆 | 市场领导者 |
| ⚔️ | 主要竞争者 |
| 🎯 | 新晋挑战者 / 策略建议 |
| 💡 | 差异化机会 |
| ⚠️ | 风险提示 |

## 竞争壁垒判断基准

| 维度 | 低壁垒 | 中等壁垒 | 高壁垒 |
|------|--------|---------|--------|
| 评价数（Top10均值）| < 500条 | 500–3,000条 | > 3,000条 |
| A+ 覆盖率 | < 50% | 50–80% | > 80% |
| 视频覆盖率 | < 30% | 30–70% | > 70% |
| 新品占比（<1年）| > 40% | 20–40% | < 20% |
| 品牌集中度（Top3）| < 30% | 30–60% | > 60% |

## 内置参考基准（半自动咖啡机 · US · 2026.03）

Skill 内置了本次真实数据的市场基准，分析其他类目时可自动对比参考：

- **市场规模**：月销 46,456 单 / 月销售额 $12,164,008
- **评价壁垒**：Top10 均 6,332 条，进入前 20 需约 2,000–3,000 条
- **新品渗透**：Top50 中 48% 为新品（上架<1年），市场仍处扩张期
- **CN卖家占比**：44% 款数 / 47.4% 销量 / 22.8% 销售额（均价拉低利润）
- **Listing 门槛**：A+ 覆盖 100%，视频覆盖 90%，LQS 均值 9.5
